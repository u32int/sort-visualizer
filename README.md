### sort-visualizer

Various sorting algorithms visualized.

Currently included algorithms:

- Bubble Sort
- Quick Sort (LR ptr)
- Insertion Sort

Dependencies:

- SDL2
- SDL2_ttf
- SDL2_gfx
- CMake

Build:

```bash
mkdir build && cd build && cmake .. && make
```

Run:

```bash
./sdlsort
```
